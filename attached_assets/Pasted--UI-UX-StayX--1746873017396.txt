أريد منك بناء واجهة وتصميم UI/UX كامل لمنصة ويب باسم "StayX"، وهي منصة مستقبلية موجهة لجمهور يهتم بالتكنولوجيا، الذكاء الاصطناعي، والعملات الرقمية. اتبع التفاصيل التالية بدقة:

1. **النمط العام (Theme)**:
   - الوضع الليلي (Dark Mode) هو الأساسي.
   - اللون الأساسي: الأخضر النيون (#39FF14)
   - الألوان الثانوية: الأسود الداكن (#0D0D0D)، الرمادي (#A9A9A9)، الأبيض (#FFFFFF)
   - الخط المستخدم: Sans-serif، ويفضّل Poppins أو Inter، عريض للعناوين، وعادي للنصوص.

2. **الشعار (Logo)**:
   - تصميم حرف X بأسلوب هندسي مستقبلي، نظيف وحاد، يعكس التكنولوجيا والتقاطع بين الأفراد والذكاء الاصطناعي.
   - يوضع في الزاوية العليا اليسرى في جميع الصفحات، باللون الأخضر النيون.

3. **الصفحات الأساسية**:
   - **الصفحة الرئيسية (Landing Page)**:
     - قسم ترحيبي فيه العبارة: “ابق متصلًا، ابق متقدمًا.”
     - زر دعوة لاتخاذ إجراء: “ابدأ الآن”
     - قسم الخدمات القادمة (أيقونات عليها "قريبًا")
     - قسم الرؤى المدعومة بالذكاء الاصطناعي (مع رسم بياني)

   - **صفحة التسجيل / تسجيل الدخول**:
     - زر تسجيل دخول بواسطة Google
     - خيار دخول كزائر
     - تصميم بسيط وأنيق

   - **لوحة التحكم (Dashboard)**:
     - شريط جانبي: الرئيسية، المحفظة، الخدمات، المجتمع، الملف الشخصي
     - وحدات عرض: الرصيد، حالة الخدمات، مساعد الذكاء الاصطناعي
     - متجاوبة (Responsive) وتعمل على الموبايل

   - **صفحة المجتمع**:
     - واجهة دردشة بسيطة
     - إنشاء / دخول غرف ChillRoom

   - **صفحة الخدمات**:
     - بطاقات للخدمات: كونسيرج، دخول VIP، تأجير سيارات، خدمات ليلية، دعم AI
     - كل بطاقة توضح إن كانت فعالة أو "قريبًا"

4. **مكونات الواجهة (UI Components)**:
   - أزرار، نوافذ منبثقة (Modals)، حقول إدخال، بطاقات، تبويبات، إشعارات
   - جميع العناصر تدعم الوضع الليلي ولها لمسة من الأخضر النيون
   - يفضّل استخدام Tailwind CSS في البنية

5. **مبادئ التصميم (Design Principles)**:
   - التصميم يكون مستقبلي، بسيط، واحترافي
   - تركيز على الوضوح والبُعد بين العناصر
   - جميع المكونات متوافقة مع معايير الوصول WCAG 2.1
   - تفعيل التفاعلات الصغيرة (تحركات الأزرار، تأثير اللمس، التبديل)

6. **الإخراج (Output)**:
   - إما على هيئة مشروع React.js + Tailwind CSS
   - أو تصميم كامل قابل للتنفيذ في Figma مع مكونات قابلة لإعادة الاستخدام

**الهدف**: إنشاء نظام UI/UX متكامل يعكس هوية StayX كمنصة مدعومة بالذكاء الاصطناعي والعملات الرقمية، ويجذب جيل الشباب الطموح المهتم بالتكنولوجيا المستقبلية.
